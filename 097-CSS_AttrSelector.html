CSS Attributes Selector

The [attribute] selector is used to select elements with a specified attribute.

1) The following example selects all elements with a target attribute:

w3schools.com disney.com wikipedia.org

2) The following example selects all elements with a target="_blank" attribute:

w3schools.com disney.com wikipedia.org

3) The [attribute~="value"] selector is used to select elements with an attribute value containing a specified word.

4) The [attribute|="value"] selector is used to select elements with the specified attribute starting with the specified value.

Welcome

Hello world!

Are you learning CSS?

Note: The value has to be a whole word, either alone, like class="top", or followed by a hyphen( - ), like class="top-text"! NOT like "topcontent"

5) The [attribute^="value"] selector is used to select elements whose attribute value begins with a specified value.

Welcome

Hello world!

Are you learning CSS?

6) The [attribute$="value"] selector is used to select elements whose attribute value ends with a specified value.

The first div element.
The second div element.
The third div element.

This is some text in a paragraph.

7) The [attribute*="value"] selector is used to select elements whose attribute value contains a specified value.

The first div element.
The second div element.
The third div element.

This is some text in a paragraph.

8) The attribute selectors can be useful for styling forms without class or ID:

Firstname: Lastname:
