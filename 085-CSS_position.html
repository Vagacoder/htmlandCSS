2) position: relative which is relative to its normal position

3) position: fixed which is fixed position relative to the viewpoint

div tag is hanging at top right corner.

4) position: absolute which is relative to nearest positioned ancestor

Note: position: fixed is to viewpoint, while position: absolute is to ancestor.
