CSS Combinator

There are four different combinators in CSS:

  • descendant selector (space)
  • child selector (>)
  • adjacent sibling selector (+)
  • general sibling selector (~)

1) Descendant selector (" ") (all descendants)

This is immediate p inside #demo1

this is grand child of #demo1

this is not child of #demo1

2) Child selector ">" (immediate children)

this is p inside #demo2

this is a grand child of #demo2

this is another p inside #demo2

this is third p inside #demo2

this is not child of #demo2

3) Adjacent Sibling Selector "+" (immediately following)

this is adjacent sibling p before #demo3

this is child p of #demo3

this is adjacent sibling p after #demo3

4) General Sibling selector "~"
