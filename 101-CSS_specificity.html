CSS Specificity

1) same rule twice in same place, lower one works

This is h3 tag, set yellow then red.

2) same rules (3 times) in same place, the one with highest specificity works

This is div with id="a"

Memorize how to calculate specificity!

Start at 0,
  • add 1000 for style attribute,
  • add 100 for each ID,
  • add 10 for each attribute, class or pseudo-class,
  • add 1 for each element name or pseudo-element.
